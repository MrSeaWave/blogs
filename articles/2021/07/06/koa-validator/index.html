更多校验方法

  • ajv：基于JSON Schema的数据校验库

JSON Schema： JSON Schema 并不是某个库，只是一种标准，简单的来说就是通过 json 格式来描述数据，而 ajv 就是对 JSON Schema 的具体实现之一.

  • class-validator：采用注解的方式进行校验，底层使用的是老牌的校验包validator.js，配合ts语法进行使用。
  • routing-controllers ：使用ts构建的一个帮助很方便处理router的框架
