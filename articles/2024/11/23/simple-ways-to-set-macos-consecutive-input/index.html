如何让Mac长按一个按键连续输入

前言

Mac 在使用系统自带英文输入法的情况下，长按一个元音字母，系统会自动弹出元音音标提示框。

苹果「Apple」设计此功能的初衷当然是为了提高输入便利性，优化用户输入体验。但是，这会对开发人员的日常工作（Coding）带来困扰，因为，我们不能长按连续输入字符了。

macOS 系统长按元音字母弹出提示框。

Mac 长按连续输入的简单设置方法

实际上，苹果「Apple」已经在用户系统配置项「Mac OS X User Defaults System」里预留出了长按连续输入的设置选项。我们可以方便地使用 defaluts 命令行工具关闭 ApplePressAndHoldEnabled 功能，设置完成后，注销或重启使其生效即可。
